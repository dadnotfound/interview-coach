## 🎯 产品简介

**面试官培训 MVP** 是一个面向 OpenClaw 用户的 AI 面试官培训工具。通过模拟真实面试场景中的"隐藏问题"，帮助面试官提升追问技巧和洞察能力。

### 核心价值

> **让面试官在安全的 AI 环境中练习，把面试失误成本降到 0。**

### 差异化特色

1. **隐藏问题机制** ⭐⭐⭐⭐⭐
   - AI 候选人有"隐藏问题"（过度包装、离职真相、薪资期望）
   - 需要通过追问发现
   - 这是真实面试中的核心痛点

2. **多维度评估** ⭐⭐⭐⭐
   - 5 维度评估框架
   - 每次面试后生成详细报告
   - 量化进步曲线

3. **零成本 + 随时可用** ⭐⭐⭐⭐⭐
   - 不需要预约，不尴尬
   - 碎片时间练习
   - 完全免费

### 安装

```bash

git clone https://github.com/your-org/interview-coach-skill.git

# 复制到 OpenClaw workspace
cp -r interview-coach-skill ~/.openclaw/workspace/skills/
```

## 🤝 贡献指南

### 如何贡献？

**方式 1：提交新问题**
1. Fork 本仓库
2. 在 `assets/questions/` 中添加问题
3. 提交 Pull Request

**方式 2：提交 Issue**
1. 点击 "New Issue"
2. 选择 "提交新问题" 模板
3. 填写表单

**方式 3：改进现有问题**
1. 找到需要改进的问题
2. 提出改进建议
3. 等待审核

### 贡献规范

#### ✅ 好的问题应该：

1. **具体明确** - 不是泛泛而谈
2. **可量化** - 可以评估回答质量
3. **真实场景** - 来自真实面试
4. **行业相关** - 和岗位 JD 高度相关
5. **符合 STAR** - 可以追问具体行为

#### ❌ 不好的问题：

1. **太泛泛** - "你有什么优势？"
2. **涉及隐私** - "你结婚了吗？"
3. **无法量化** - "你是一个有创意的人吗？"
4. **不相关** - "你平时喜欢做什么运动？"
5. **过于简单** - "你会用 Excel 吗？"
