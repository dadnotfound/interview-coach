# 行为准则

## 我们的承诺

为了营造开放和友好的环境，我们作为贡献者和维护者承诺：让每个人都能参与到这个项目中。

## 我们的标准

**积极行为包括：**
- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受的行为包括：**
- 使用性化语言或图像
- 恶意攻击或侮辱性评论
- 骚扰或恶意行为
- 未经许可发布他人私人信息
- 其他不专业或不恰当的行为

## 我们的责任

项目维护者负责阐明可接受行为的标准，并对任何不可接受的行为采取适当和公平的纠正措施。

## 范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时。

## 执行

项目团队可以联系作者 [984525027@qq.com](mailto:984525027@qq.com) 来举报滥用、骚扰或其他不可接受的行为。

所有投诉都将被审查和调查，并将产生被认为必要和适合情况的回应。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献。
